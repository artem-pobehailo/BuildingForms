# Building Forms

A responsive registration form built with HTML5 and Tailwind CSS.

## Features

- **User Registration Form** with the following fields:
  - Full Name (required)
  - Email (required)
  - Password & Confirm Password (required, with validation)
  - Gender Selection (radio buttons)
  - Age Input (numeric, 1-120 range)
  - Course Selection (Front-End or Back-End)
  - Full Address (textarea)
  - Terms & Conditions Agreement (checkbox)

- **Form Validation**:
  - Password confirmation validation on submit
  - HTML5 built-in validation for required fields
  - Email format validation
  - Age range validation (1-120)

- **Responsive Design**:
  - Built with Tailwind CSS for modern styling
  - Centered layout with proper spacing
  - Focus ring styling for better accessibility
  - Hover effects on buttons

- **Interactive Features**:
  - Reset button to clear the form
  - Submit button with client-side password validation
  - Password mismatch alert

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Usage

Open `index.html` in a web browser to view and interact with the form.

## Language

The form is currently in Ukrainian (Українська).
